# 🚩 领地

这是个古老的插件，/res指令我们都再熟悉不过了

你可以去看他的WIKI了解更多

下面是常用的指令：（注：格式：/res <下面的指令>）

```
create - 创建领地
auto - 以你的位置为中心, 创建你能创建最大的领地
placeholders - 列出 Placeholder 列表
tpset - 设置领地的传送点
list - 显示领地列表
padd - 向领地添加玩家
pdel - 从领地删除玩家
info - 显示领地的信息
set - 为领地设置公共权限
setallfor - 设置指定玩家的所有领地的公共权限
setall - 设置所有领地的公共权限
pset - 为玩家设置指定领地的特定权限
limits - 显示你的领地限制
message - 管理加入/离开领地时的提示消息
confirm - 确认删除领地
rc - 加入领地聊天频道
flags - 领地权限列表
select - 选区命令
tp - 传送到领地
tpconfirm - 忽略传送警告
tool - 显示用于领地选区和查看领地信息的工具的名称
shop - 管理领地商店
contract - 从你面对的方向缩小领地
expand - 向你面对的方向扩展领地
subzone - 在领地中创建子领地
kick - 将玩家踢出领地
remove - 删除领地
rt - 随机传送
market - 购买, 出售和出租领地
rename - 重命名领地
renamearea - 修改领地区域名称
setmain - 设置为主领地
command - 管理领地允许或者禁止的命令
raid - 管理领地的袭击
raidstatus - 查看领地的袭击状态
current - 显示你当前所在的领地
leaveraid - 离开袭击
defend - 加入领地袭击的防守
attack - 开始对领地进行袭击
compass - 将指南针指向领地
area - 管理领地的区域
show - 显示领地的边界
bank - 管理领地银行
check - 查看权限状态
mirror - 权限克隆
give - 将你拥有的指定领地给予指定玩家
lease - 管理领地租约
unstuck - 传送到领地外面
sublist - 列出指定领地的所有子领地
listall - 列出所有的领地
material - 通过ID查看是否存在对应的材料
reset - 将领地重置为默认权限
gset - 为权限组设置领地的特定权限
lists - 预定义权限列表
lset - 更改黑名单和忽略名单选项
```
