# 💡 进服做什么

首次进入服务器，你需要输入指令进行注册

```
/register <输入密码> <再次输入密码>
```

等以后进入服务器，只需要输入

```
/login <输入密码>
```



然后你就可以使用组合键Shift+F打开菜单，开始你的生存吧。
